# 📄 Law Impact Feature Documentation

## 📌 Описание

**Law Impact Service** — модуль, который анализирует изменения законодательства и определяет, влияют ли они на конкретное дело пользователя.

Если изменение релевантно — пользователь получает уведомление с рекомендациями.

---

## 🧠 Как это работает

1. Есть дело пользователя (категория + описание)
2. Есть список изменений законодательства
3. AI анализирует каждое изменение
4. Возвращаются только релевантные изменения (`affects = true`)
